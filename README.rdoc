== README

Purpose of this application is finding of intersection diffent figures.
The application use custom gem [figures_intersections](https://github.com/vovkats/figures_intersection).

Available Figures
* Point
* Straight Line y = k*x+b
* Vertical line x = a
* Segment (need to set coords of ends)
* Rectangle (given the coordinates of its two diagonally opposite vertices)
For run application:
rails s
